# rn-ramen

A delicious component library for React Native based un styled system and inspired by Charka UI

## Installation

```sh
npm install rn-ramen
```

## Usage

```js
import RnRamen from "rn-ramen";

// ...

const result = await RnRamen.multiply(3, 7);
```
